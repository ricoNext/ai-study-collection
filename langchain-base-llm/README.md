# base-llm

大模型基础调用示例

## 开始

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## 添加环境变量

在 .env 中替换 doubao apikey

```
DOUBAO_API_KEY=xxxxx
DOUBAO_API_BASE_URL=https://ark.cn-beijing.volces.com/api/v3/
```

## 运行

打开 [http://localhost:3000](http://localhost:3000) 查看结果。
